# 🎓 SkillSphere

## Purpose

SkillSphere is a modern online learning platform where users can explore courses, view course details, and manage their profiles through secure authentication.

## Live URL

https://skill-sphere-psi-tawny.vercel.app/

## Key Features

- Email & Password Authentication
- Google Login Authentication
- Protected Course Details Page
- Browse All Courses
- Search Courses by Title
- Popular Courses Section
- Learning Tips Section
- Top Instructors Section
- New Releases Section
- User Profile Page
- Update Profile Information
- Responsive Design
- Toast Notifications
- Custom 404 Page

## NPM Packages Used

- better-auth
- swiper
- react-hot-toast
- react-icons
- tailwindcss
- daisyui
